# ANYTYPE 🧠📝

**تسجيل المحادثات وتلخيصها تلقائياً للمراجعة والتحسين المستمر.**

إضافة ذكية (Agent Skill) تقوم بتسجيل وتلخيص كل محادثة تلقائياً، مما يسمح بالرجوع إلى أي جلسة سابقة واستئناف العمل من حيث توقف.

📂 المشروع منظم تحت `~/hermes/anytype/` — جميع المشاريع داخل `~/hermes/`.

## الميزات

- ✅ **تسجيل تلقائي:** في نهاية كل جلسة، يتم إنشاء ملخص منظم
- ✅ **استئناف ذكي:** عند قول "ارجع" أو "اكمل ماكنا نفعله"، يراجع الملخصات السابقة
- ✅ **تحسين مستمر:** يتم تسجيل نقاط التحسين وتطبيقها تلقائياً
- ✅ **فهرس مركزي:** كل الجلسات مفهرسة وسهلة التصفح

## كيف يعمل

1. يتحدث المستخدم مع الذكاء الاصطناعي كالمعتاد
2. في نهاية كل جلسة، يُطلب من المستخدم تأكيد التسجيل
3. يتم إنشاء ملخص منظم وحفظه في `sessions/`
4. عند الحاجة، يمكن استدعاء أي جلسة سابقة

## البنية

```
~/hermes/
└── anytype/                  # مشروع ANYTYPE داخل hermes
    ├── README.md             # هذا الملف
    ├── INDEX.md              # فهرس جميع الجلسات
    ├── sessions/             # ملخصات الجلسات
    │   └── YYYY-MM-DD_summary.md
    └── improvements/         # اقتراحات التحسين
        └── ...
```

## الاستخدام

- **"سجل هده الجلسة"** — تسجيل الجلسة الحالية
- **"ارجع"** — عرض آخر الجلسات
- **"اكمل ماكنا نفعله"** — استئناف آخر جلسة
- **"حسن ANYTYPE"** — إضافة اقتراح تحسين
